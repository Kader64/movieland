# 🎬 MovieLand

**MovieLand** to prosta aplikacja webowa stworzona w **React**, umożliwiająca wyszukiwanie filmów przy użyciu zewnętrznego API – **OMDb API**.  
Aplikacja pozwala użytkownikowi dynamicznie przeszukiwać bazę filmów oraz wyświetlać wyniki w formie estetycznych kart.

---

## 🌐 Opis projektu

MovieLand to wyszukiwarka filmów działająca w oparciu o publiczne API:

- 🔎 Wyszukiwanie filmów po tytule  
- 🎞️ Wyświetlanie plakatu filmu  
- 📅 Informacja o roku produkcji  
- 🎭 Typ produkcji (movie, series, episode)  
- ⚡ Dynamiczne aktualizowanie wyników podczas wpisywania tekstu  

Domyślnie po uruchomieniu aplikacja wyszukuje filmy związane z hasłem **"Batman"**.

---

## 🛠️ Technologie

- ⚛️ React (Hooks: `useState`, `useEffect`)
- 🌐 Fetch API
- 🎨 CSS
- 🔑 OMDb API (http://www.omdbapi.com)
